- Database : MySQL
- DevOps : Docker, AWS EC2, GitHub Action, Git Hooks

### ⚙️ 설치 및 실행 방법
- 환경변수는 각 경로에 있는 `.env.example` 파일을 참고하여 작성

**프론트엔드**
```
cd FE/
npm install
npm run dev
```

**백엔드**(docker desktop 실행 후 진행)
```
cd BE/
./gradlew bootrun
```
